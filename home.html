Interrogare il Knowledge Graph: Tre Esempi di Query

Queste query dimostrano come possiamo esplorare il Knowledge Graph di **Arco Beni Culturali** per trovare informazioni dettagliate sul Bacco di Caravaggio e opere simili. Clicca sul link per vedere un risultato di esempio su Arco.

Query 1: Ricerca per Autore e Titolo

Cerca opere di "Caravaggio" con il titolo "Bacco".


PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT ?opera ?titolo
WHERE {
  ?opera a arco:OperaArte ;
         arco:hasAutore ?autore ;
         arco:hasTitolo ?titolo .
  ?autore rdfs:label "Caravaggio" .
  FILTER (contains(lcase(?titolo), "bacco"))
}
LIMIT 10

Query 2: Opere a Tema Mitologico

Trova tutte le opere di Caravaggio con soggetti mitologici.


PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
PREFIX skos: <http://www.w3.org/2004/02/skos/core#>

SELECT ?opera ?titolo ?soggetto
WHERE {
  ?opera a arco:OperaArte ;
         arco:hasAutore ?autore ;
         arco:hasTitolo ?titolo ;
         arco:hasSoggetto ?soggettoURI .
  ?autore rdfs:label "Caravaggio" .
  ?soggettoURI skos:prefLabel ?soggetto .
  FILTER (regex(?soggetto, "mitologic", "i"))
}
LIMIT 10

Query 3: Opere di Autori Nati nel Periodo di Caravaggio

Scopri opere di artisti attivi nello stesso periodo di Caravaggio.


PREFIX arco: <https://w3id.org/arco/ontology/arco/>
PREFIX schema: <http://schema.org/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?opera ?titolo ?altroAutore
WHERE {
  ?opera a arco:OperaArte ;
         arco:hasAutore ?altroAutoreURI ;
         arco:hasTitolo ?titolo .
  ?altroAutoreURI rdfs:label ?altroAutore .

  ?caravaggioURI rdfs:label "Caravaggio" ;
                  schema:birthDate ?nascitaCaravaggio .

  ?altroAutoreURI schema:birthDate ?nascitaAltroAutore .

  FILTER (?nascitaAltroAutore >= "1570-01-01"^^xsd:date && ?nascitaAltroAutore <= "1620-12-31"^^xsd:date)
  FILTER (?altroAutore != "Caravaggio") # Escludi Caravaggio stesso
}
LIMIT 10
