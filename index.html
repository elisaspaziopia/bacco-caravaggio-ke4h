Query SPARQL per il Bacco di Caravaggio

Esplora le query utilizzate per identificare e analizzare l'opera "Bacco" di Caravaggio all'interno del Knowledge Graph di Arco Beni Culturali.

Query Base - Identificazione Opera

Query fondamentale per individuare l'opera "Bacco" di Caravaggio nel database Arco, recuperando informazioni essenziali come titolo, autore e classificazione.

PREFIX arco: <https://w3id.org/arco/ontology/arco/> SELECT ?opera ?titolo ?autore WHERE { ?opera a arco:CulturalProperty ; arco:hasTitle ?titolo ; arco:hasAuthor ?autore . FILTER(contains(lcase(?titolo), "bacco")) FILTER(contains(lcase(?autore), "caravaggio")) }
Visualizza Risultati

Query Avanzata - Dettagli Tecnici

Query approfondita per ottenere informazioni tecniche dettagliate sull'opera, inclusi materiali, dimensioni, tecniche pittoriche e stato di conservazione.

PREFIX arco: <https://w3id.org/arco/ontology/arco/> SELECT ?opera ?materiale ?dimensioni ?tecnica WHERE { ?opera arco:hasTitle ?titolo ; arco:hasMaterial ?materiale ; arco:hasDimension ?dimensioni ; arco:hasTechnique ?tecnica . FILTER(contains(lcase(?titolo), "bacco")) }
Visualizza Risultati

Query Contestuale - Relazioni Storiche

Query complessa per esplorare il contesto storico-artistico dell'opera, collegando informazioni su committenza, ubicazione storica e relazioni con altre opere.

PREFIX arco: <https://w3id.org/arco/ontology/arco/> SELECT ?opera ?ubicazione ?commissione ?periodo WHERE { ?opera arco:hasTitle ?titolo ; arco:hasLocation ?ubicazione ; arco:hasCommission ?commissione ; arco:belongsToPeriod ?periodo . FILTER(contains(lcase(?titolo), "bacco")) }
