Exploring Bacchus by Caravaggio: how much does ArCo tell us?

Our project is based on enriching the content of the Italian ArCo database concerning Caravaggio's artwork Bacchus. This painting is part of the Uffizi Gallery in Florence, contributing, together with the other works of art, to making it one of the most visited museums in the world.

To find it on ArCo, and note the missing information, that is, gaps, we developed some SPARQL queries. Then, to add said information, we made use of several LLMs (Large Language Models) using prompting techniques. We then transformed the outputs obtained into triples that could enrich the ArCo database.

Please scroll down for more detailed information about our research.

Methodology

First, after choosing the subject to search on ArCo, we used these queries, already containing the main SPARQL keywords.

Query 1: Localizzazione e Dettagli Principali

PREFIX arco: <https://w3id.org/arco/ontology/arco/> PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> PREFIX dc: <http://purl.org/dc/elements/1.1/> PREFIX foaf: <http://xmlns.com/foaf/0.1/> SELECT DISTINCT ?opera ?titolo ?autoreLabel WHERE { ?opera a arco:HistoricOrArtisticProperty . { ?opera rdfs:label ?titolo . FILTER(REGEX(LCASE(STR(?titolo)), "bacco", "i")) } UNION { ?opera dc:title ?titolo . FILTER(REGEX(LCASE(STR(?titolo)), "bacco", "i")) } OPTIONAL { { ?opera dc:creator ?autore . } UNION { ?opera foaf:maker ?autore . } OPTIONAL { ?autore rdfs:label ?autoreLabel . FILTER(REGEX(LCASE(STR(?autoreLabel)), "caravaggio", "i")) } } } ORDER BY ?titolo LIMIT 200

Questa query fondamentale è progettata per localizzare rapidamente l'opera "Bacco" nel Knowledge Graph di Arco. Non solo recupera il titolo e l'autore, ma cerca anche di stabilire un link diretto all'autore per poter esplorare ulteriori dettagli biografici o altre opere a lui attribuite, fornendo un punto di partenza per l'analisi. **Puoi aggiungere qui una descrizione più dettagliata o i risultati chiave di questa query.**

Visualizza Risultati su Arco

Dopo aver identificato l'opera principale, abbiamo approfondito la ricerca per estrarre dettagli specifici che ci avrebbero aiutato a comprendere meglio le lacune informative nel Knowledge Graph di ArCo.

Query 2: Dettagli Tecnici e Localizzazione dell'Opera

PREFIX arco: <https://w3id.org/arco/ontology/arco/> PREFIX schema: <http://schema.org/> PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> SELECT ?opera ?materiale ?dimensioni ?tecnica ?ubicazioneNome WHERE { ?opera arco:hasTitle ?titolo ; arco:hasMaterial ?materialeUri ; arco:hasDimension ?dimensioniUri ; arco:hasTechnique ?tecnicaUri ; schema:location ?ubicazione . OPTIONAL { ?materialeUri rdfs:label ?materiale . } OPTIONAL { ?dimensioniUri rdfs:label ?dimensioni . } OPTIONAL { ?tecnicaUri rdfs:label ?tecnica . } OPTIONAL { ?ubicazione rdfs:label ?ubicazioneNome . } FILTER(contains(lcase(?titolo), "bacco")) } LIMIT 200

Questa query si concentra sui dettagli fisici dell'opera, come il materiale, le dimensioni e la tecnica utilizzata, oltre alla sua ubicazione attuale. È fondamentale per comprendere il contesto materiale e la conservazione del Bacco, fornendo dati specifici che possono essere arricchiti.

**Spiegazione della query:**
- `DISTINCT`: Evita risultati duplicati.
- `FILTER (REGEX(LCASE(STR(?titolo)), "bacco", "i"))`: Limita i risultati al titolo "Bacco" (case-insensitive).
- `UNION`: Combina risultati da diverse proprietà del titolo o dell'autore.
- `OPTIONAL`: Rende non obbligatoria la presenza di determinate informazioni (es. autore), per non escludere risultati validi.
- `ORDER BY`: Ordina i risultati.
- `LIMIT`: Limita il numero massimo di risultati.
