Query SPARQL per il Bacco di Caravaggio

Query 1: Ricerca Base

Query principale per identificare l'opera "Bacco" di Caravaggio nel database di Arco Beni Culturali.

PREFIX arco: <https://w3id.org/arco/ontology/arco/>
SELECT ?opera ?titolo WHERE {
  ?opera arco:hasTitle ?titolo .
  FILTER(contains(?titolo, "Bacco"))
}
