Exploring Bacchus by Caravaggio: how much does arco tell us?

Our project is based on enriching the content of the Italian ArCo database concerning Caravaggio's artwork Bacchus. This painting is part of the Uffizi Gallery in florence, contibuiting, together with the other works of art, to making it one of the most visited museums in the world.

To find it on ArCo, and note the missing information, that is, gaps, we developed somme SPARQL queries. Then, to add said information, we made use of several LLMs (Large Language Models) using prompting techniqes. We then transformed the outputs obtained into triples that could enrich the ArCo database. Please scroll down for more detailed information about our research.

Methodology

Esplora le query utilizzate per identificare e analizzare l'opera "Bacco" di Caravaggio all'interno del Knowledge Graph di Arco Beni Culturali. Ciascuna query è accompagnata da uno spazio per descrizioni dettagliate sulla sua utilità e sui risultati attesi.

Query 1: Identificazione Base e Contesto Autore

Questa query fondamentale è progettata per localizzare rapidamente l'opera "Bacco" nel Knowledge Graph di Arco. Non solo recupera il titolo e l'autore, ma cerca anche di stabilire un link diretto all'autore per poter esplorare ulteriori dettagli biografici o altre opere a lui attribuite, fornendo un punto di partenza per l'analisi. **Puoi aggiungere qui una descrizione più dettagliata o i risultati chiave di questa query.**

Visualizza Risultati
PREFIX arco: <https://w3id.org/arco/ontology/arco/> PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#> SELECT ?opera ?titolo ?autoreNome ?autoreUri WHERE { ?opera a arco:CulturalProperty ; arco:hasTitle ?titolo ; arco:hasAuthor ?autoreUri . ?autoreUri rdfs:label ?autoreNome . FILTER(contains(lcase(?titolo), "bacco")) FILTER(contains(lcase(?autoreNome), "caravaggio")) }

Query 2: Dettagli Tecnici e Localizzazione dell'Opera

Questa query si concentra sui dati più specifici dell'opera "Bacco", recuperando informazioni cruciali come i materiali utilizzati, le dimensioni esatte e la tecnica pittorica. Inoltre, individua l'attuale ubicazione dell'opera all'interno di una collezione o di un museo, offrendo un quadro completo sulle sue caratteristiche fisiche e la sua collocazione attuale. **Utilizza questo spazio per descrivere l'importanza di questi dettagli o esempi di come le LLMs potrebbero arricchire queste informazioni.**

Visualizza Risultati
PREFIX arco: <https://w3id.org/arco/ontology/arco/> PREFIX schema: <http://schema.org/> SELECT ?opera ?materiale ?dimensioni ?tecnica ?ubicazioneNome WHERE { ?opera arco:hasTitle ?titolo ; arco:hasMaterial ?materiale ; arco:hasDimension ?dimensioni ; arco:hasTechnique ?tecnica ; schema:location ?ubicazione . ?ubicazione rdfs:label ?ubicazioneNome . FILTER(contains(lcase(?titolo), "bacco")) }
